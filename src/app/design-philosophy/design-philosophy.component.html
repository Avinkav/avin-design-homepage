Design Philosophy

Mobile First

Active mobile devices heavily outnumber PCs. However, Mobiles do not match PCs in compute power or screen size. A well designed website takes this into consideration. The layout is first optimized for mobile and then built up to fit bigger screens, leading to a pleasant and postive experience for mobile users.

Responsive Design

Display dimensions vary between devices and change with orientation. User interface layouts suit the dimensions in use and also adapt as dimensions change. Details are revealed as dimensions increase and summarised as they shrink, all in one fluid flowing design.

Rich Interactive UX

Typography, Color and Animations blend in seemlessly to provide a rich interactive user experience.

Universal

Use of open source cross-platform frameworks and tools ensure portability and availabilty across major platforms for both developers and end users.

Elegant Coding

Designed applications are durable and reliable due to precise, concise coding and industry best practices.
